## ğŸ—‚ï¸ Project Structure

The project follows the Next.js App Router architecture.

- The app directory contains application routes, pages, and global styles
- Reusable UI components with their own CSS Modules are stored in the components directory
- The lib folder contains API logic and global state
- Shared TypeScript types are placed in the types directory
- Static assets such as images and icons are located in the public folder
